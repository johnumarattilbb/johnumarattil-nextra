# I'm Johnu Marattil

I'm a Product Designer currently employed at Harrison Healthcare. I have a passion for 
solving problems. My joy comes from applying design thinking methodology to transform complex 
user journeys into easily comprehensible user flows in products.  I excel in user research, 
copywriting, wireframing, prototyping, visual design, interaction design, and usability testing. 
Additionally, I excel in technologies like React, NextJs, HTML, CSS, JavaScript, 
WebGL, Lottie, Rive animations, and have proficiency in both mobile and web development.
